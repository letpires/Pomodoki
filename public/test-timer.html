How it works

When you start a timer:

  • A timer overlay will appear in the top-right corner of this page
  • The timer will be draggable - you can move it around
  • You can close it with the Ã— button
  • The timer will show the remaining time in MM:SS format
  • In the last 30 seconds, it will pulse to alert you
  • The timer will automatically disappear when time runs out
